ESP32 Hydraulic Excavator Kit

Year: 2023 (Year 11 Systems Engineering)

Overview

Designed and built a hydraulically actuated excavator controlled via Bluetooth using an ESP32 microcontroller. The project aimed to create an educational STEM kit demonstrating hydraulics, electronics, and embedded control.

Problem

Existing STEM kits lacked strong hydraulic integration and microcontroller-based control systems. The goal was to design a functional excavator capable of lifting 100g to a height of 5+ cm.

Design Approach

  • ESP32 microcontroller
  • MX1508 motor drivers
  • N20 geared motors (200 RPM)
  • Peristaltic pumps for hydraulic actuation
  • 5mL & 3mL syringe-based hydraulic joints
  • Bluetooth control via Dabble mobile app
  • LiFePO₄ 32650 battery cells

Testing & Results

Successfully lifted 100g load to a height of 7 cm. Initial syringe slippage was resolved using permanent mounting. Excavator maintained functional tank-style steering control.

Challenges

  • Syringe backflow issues
  • Wheel dimension inaccuracies after 3D printing
  • Traction limitations
  • Water leakage between connections

Future Improvements

  • Add first-person camera via ESP32
  • Redesign wheel shafts for tighter fit
  • Improve water containment system
  • Enhance chassis aesthetics and durability
